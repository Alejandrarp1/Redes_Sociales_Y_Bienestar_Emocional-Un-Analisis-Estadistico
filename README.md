# Redes_Sociales_Y_Bienestar_Emocional-Un-Analisis-Estadistico
Proyecto de consultoría estadística que analiza la relación entre el uso de redes sociales y el bienestar emocional. Mediante análisis exploratorio, regresión logística multinomial y machine learning, se identifica emociones dominantes y patrones de comportamiento digital asociados, aportando evidencia para promover hábitos digitales saludables.
